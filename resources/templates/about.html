About

This site allows you to submit domains for crawling of their Ads.txt files. When a valid Ads.txt file is present this system will parse it's contents and save the resulting records to it's database. If the system has valid records for a domain the the number of records will be displayed on the domains page and when clicked will show the associated records.

This project was developed by Brad Lucas. The first part of the system was developed as a library to crawl sites. It's source code is available in GitHub. Also, Brad has a series of blog posts describing the development of this project. See the following links for the GitHub repository, blog posts and more information on the Ads.txt specification.
